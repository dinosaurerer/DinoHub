# 🌐 Dino Tools Navigation

一个简洁美观的个人工具导航页面，为根域名 `094694.xyz` 设计的统一入口，整合了我使用二级域名创建的各种实用工具网站。

## ✨ 项目特色

- 🎨 **现代化设计** - 采用卡片式翻转交互，视觉体验出色
- 📱 **响应式布局** - 完美适配桌面端、平板和移动设备
- ⚡ **快速访问** - 一站式访问所有个人工具项目
- 🎬 **动态效果** - 每个卡片配有相关主题的预览视频

## 🛠️ 工具集合

### 核心工具
- **DinoSpace** - 个人网盘，安全的云存储解决方案
- **DinoBlog** - 个人博客，技术分享与生活记录
- **DinoImBed** - 图床服务，便捷的图片托管
- **DinoMail** - 临时邮箱，保护隐私的邮件服务

### 实用工具
- **DinoShort** - 短链接生成器，简化长网址分享
- **DinoPayQrcode** - 聚合码生成，统一收款码管理
- **DinoSubs** - 订阅提醒，个人日程管理助手

### 特色项目
- **DinoAnniv** - 纪念日网站，记录重要时刻
- **Dino-love** - HTML特效展示，创意动画效果
- **DinoImBox** - 图床旧版，历史版本保留

## 🚀 快速开始

1. 访问主站：https://094694.xyz
2. 浏览工具卡片，点击查看详情
3. 悬停卡片可查看动态预览效果
4. 点击"访问网站"按钮直达对应工具

## 🎯 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **样式**: 自定义CSS，响应式网格布局
- **交互**: CSS3动画，卡片翻转效果
- **部署**: 静态网站托管
